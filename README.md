Project Overview

This project demonstrates the use of Python lists and their built-in methods through a fun Justice League-themed exercise. It covers common list operations such as adding, removing, inserting, replacing, sorting, and counting elements. The program displays the list after each operation to help understand how list manipulation works in Python.

Objectives

- Create and manipulate Python lists.
- Calculate the number of elements using "len()".
- Add new elements using "extend()".
- Remove and insert elements using "remove()" and "insert()".
- Replace an entire list with new values.
- Sort a list alphabetically using "sort()".
- Access list elements using indexing.

Features

- Counts the total number of Justice League members.
- Adds new superheroes to the team.
- Changes the team leader by rearranging the list.
- Separates specific members by repositioning another hero.
- Replaces the existing team with a new lineup.
- Sorts the final list alphabetically and identifies the new leader.

Technologies Used
